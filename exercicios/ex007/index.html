Principais tags para a formatação de textos

Negrito / Destaque

Nessa frase temos um negrito usando uma tag b (não semântica)

Nesta frase temos um termo em destaque usando a tag STRONG (semântica).

italico / Ênfase

Nesta frase temos um termo em italico usando a tag i (não semântica)

Nesta tag temos um termo em ênfase usando a tag EM (semântica)

Texto marcado / Marca texto

Podemos criar também um texto marcado usando a tag MARK

E no outro parágrafo, temos outro texto marcado no final no final

Texto grande e pequeno

Estamos criando nesse parágrafo, texto grande (não semântica) e texto pequeno (semântica)

Texto deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não deve ser considerado

Texto inserido

Podemos marcar um texto como inserido para da ênfase e indicar que ele foi adicionado depois (semântica)

Existe também o sublinhado com a tag U (não semântica)

Texto sobrescrito

Para inserir coisas do tipo: x2+3=

Texto subscrito

Para inserir coisas do tipo: H2O
